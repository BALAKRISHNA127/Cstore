# CStores – E‑Commerce Website

CStores is a front‑end e‑commerce website inspired by Flipkart’s UI/UX. It is built using **HTML, CSS, and vanilla JavaScript** and demonstrates core e‑commerce features such as product listing, category filtering, search, sorting, and a client‑side shopping cart with persistence using `localStorage`.

---

## ✨ Features

* 🛍️ Product listing with mock data
* 🔎 Search products by name
* 🧩 Filter products by category
* 🔃 Sort products by price and rating
* 🛒 Add to cart, update quantity, remove items
* 💾 Cart data persisted using `localStorage`
* 📱 Responsive layout (desktop, tablet, mobile)
* 🎨 Flipkart‑style UI with CSS variables and modern design
* 📦 Cart drawer with subtotal, discount, and total calculation

## 🚀 Getting Started

You can run this project locally without any build tools.

### Steps:

1. Clone the repository

   ```bash
   git clone https://github.com/your-username/cstores.git
   ```

2. Open the project folder

   ```bash
   cd cstores
   ```

3. Open `index.html` in your browser

   * Double‑click the file, **or**
   * Use a live server extension (recommended)

---

## 🧠 How It Works

* Products are stored as a JavaScript array in `script.js`.
* Categories are generated dynamically from product data.
* Search, filter, and sort update the product grid in real time.
* Cart data is stored as an object and saved to `localStorage`.
* Cart UI updates automatically when items are added or modified.
